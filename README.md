Dojo Tic-Tac-toe IA
======================

### Demarrer le projet
* Importer le projet `git clone git@github.com:landschootl/tic-tia-dojo.git`.
* Entrer dans le dossier `cd tic-tia-dojo`
* Ouvrir le fichier 'index.html'.
